Lay-outs

In Powerall Connect worden er diverse lay-outs gebruikt. o.a. voor overzichten, lijsten, bonnen, facturen, formulieren etc. Het is mogelijk om per programma meerdere lay-outs op te geven. Standaard wordt layout 'A-Standaard layout' gebruikt.

In het menu Formulierlayouts (BP) kunnen deze onderhouden, geïmporteerd of geëxporteerd worden.

Opmerking: De papierhoogte, breedte en marges worden in inch weergegeven. De standaard layouts A t/m D hebben standaard font Segoe UI.

De volgende onderwerpen worden behandeld:

  • Lay-out aanmaken of kopiëren
  • Lay-out wijzigen
  • Lay-out verwijderen
  • Lay-out importeren
  • Lay-out exporteren
  • Word layouts

Let op!  Het muteren van lay-outs A t/m D is een 'Beveiligde functie', hiervoor is een wachtwoord nodig.
