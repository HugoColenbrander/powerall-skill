Overzicht machine website gegevens

Als een machine op een of meerdere websites staat / gepubliceerd is, kunnen de gegevens bekeken worden. Om dit te doen, doorloopt u de volgende stappen:

  1. Ga naar Opvragen machines  (MO) of F8
    of

    Ga naar Onderhoud machines (MBB).

  2. Ga naar tabblad Website.

    • Bij machine administratie 2.0 is dit tabblad Commercieel.

  3. Kies voor de knop  Overzicht.

    • Het scherm Overzicht website gegevens verschijnt.

  4. Om de gegevens (incl. eventuele foto's) af te drukken of te mailen:

    1. Kies voor de knop  Print.

    2. Het overzicht verschijnt op het scherm.

 

FAQ

Hoe kan ik deze website gegevens wijzigen?

Om bepaalde website gegevens te wijzigen, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud machines (MBB).

  2. Kies voor de knop  Overzicht.

  3. Ga naar tabblad Website.

  4. Wijzig de gegevens / foto's

  5. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

Wat doe ik bij de melding 'Standaard website koppeling niet ingesteld!'?

Waarschijnlijk is er iets niet goed ingesteld.

  • Neem hiervoor contact op met de helpdesk om dit op te lossen.

Welke website wordt standaard weergegeven?

In de kop wordt standaard de omschrijving van website code BS weergegeven, zie (MEB).
