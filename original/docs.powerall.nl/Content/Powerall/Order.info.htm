Verzendinformatie verkooporder / Extra info

Bij een verkooporder of weborder is het mogelijk om verzendinformatie toe te voegen of te wijzigen.

Klik hier, voor meer informatie over dit onderwerp.

Extra informatie kan ook ingegeven worden, vanaf versie 3.30 bij een:

  • Bon

  • Offerte

  • Werkorder, Tabblad Algemeen 

Als de order wordt vrijgegeven, gaat de extra informatie ook mee.

 

Om dit te doen, doorloopt u de volgende stappen:

  1. Kies voor de knop  (notitie).

    • Het scherm Verzendinformatie verkooporder verschijnt.

    • Het scherm Extra informatie verschijnt, vanaf versie 3.30.

  2. Vul of wijzig de gegevens.

    Klik hier voor meer informatie over Webshop afhaalpunt.

    Bij de webshop i.c.m. een afhaalpunt / SendCloud kan het afhaalpunt aan Powerall Connect doorgegeven worden.

    • Bij een weborder kan deze informatie automatisch ingevuld worden.

    • Bepaalde gegevens van deze verzendinformatie kunnen afgedrukt worden op de PostNL verzendlabel, zie ook Hoe, wat kan ik ... m.b.t. PostNL?
  3. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

 

FAQ

  • Als er informatie ingevuld is, wordt de notitie knop rood.
  • Als de OrderReference leeg is, wordt de referentie van de order gevuld voor Peppol.
  • Als je naar een Duits Leitweg-ID een factuur gestuurd wordt, wordt de BuyerReference niet meegenomen naar Peppol.

  • Als de order wordt vrijgegeven, gaat deze info mee, bijv. naar de bonnen, vanaf versie 3.30.

Hieronder een uitleg van de velden van scherm Verzendinformatie verkooporder:

 

Kop  
Track & Trace Het Track & Trace nummer m.b.t. VO.
Bezorginfo Bezorg informatie verkooporder voor vervoerders, gebruikt bij PostNL.
 
Velden:
- Note
- AccountingCost
- BuyerReference
- OrderReference 		Selecteer de velden die van toepassing zijn en vul deze in, vanaf versie 3.30.
- Note: vrije ingave bijvoorbeeld kenteken (zonder streepjes) + ‘job’ + jobnummer
- AccountingCost: Financiële boekings info (Peppol / routecode)
- BuyerReference: Klant referentie / jobnummer
- OrderReference: Order referentie / inkoopordernummer
 
E-mailadres De e-mailadres m.b.t. de verkooporder.
Telefoonnummer Het betreffend telefoonnummer.
 
Service Point Het service point adres bijv. van PostNL instelling.
