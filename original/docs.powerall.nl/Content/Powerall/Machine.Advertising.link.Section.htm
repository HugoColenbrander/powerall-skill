Hoe pas ik bij een machine rubriek op website?

Om de machines éénmalig / vooralsnog naar een andere /de juiste rubriek te verplaatsen, doorloopt u de volgende stappen:

  1. Log in op het addons.powerall.app.

  2. Selecteer de administratie / organisatie.
    • Het Dashboard verschijnt.
  3. Ga naar Advertenties.

    • Oud: Ga naar Advertentie / koppeling > Machines.

  4. Selecteer of zoek het servicenummer op.

  5. Klik deze open.

  6. Klik op betreffende website.

  7. Klik bij Gegevens op de link Link naar instellingen (niet op Wijzigen).

  8. Vervolgens verschijnt het volgende:

    • Wijzig de categorie onder het kopje Nieuwe waarde.

Opmerking: Machines in de juiste rubriek worden sneller gevonden.
