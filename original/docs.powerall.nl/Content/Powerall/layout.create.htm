Lay-out aanmaken of kopiëren

Om een nieuwe layout toe te voegen, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud layouts (BPO).

  2. Selecteer bij programma het juiste overzicht/factuur waarvoor deze lay-out geldt.

  3. Vul bij lay-out een nieuwe letter in.

    • De knop Opvoer wordt actief.

  4. Kies voor de knop  Opvoer.

    • Het scherm Kopieren van layout verschijnt.

    1. Vul het programma en de lay-out in en vink (alles) aan, wat gekopieerd moeten worden.

    2. Kies voor de knop  Kopiëren.

  5. Bij tabblad Rapport pas de rapportnaam eventueel aan.

    • Afdruk stand 1 = horizontaal
    • Afdrukstand 2 = verticaal bijv. voor boomlabels
    • NB De papier/marge gegevens worden bij het opslaan gevuld.

  6. Wijzig op tabblad Details de gegevens.

    • Voor meer informatie zie Lay-out wijzigen.

    Opmerking: De papierhoogte, breedte en marges worden in inch weergegeven. De standaard layouts A t/m D hebben standaard font Segoe UI.

  7. Kies voor de knop  Voorbeeld om het rapport te bekijken. Indien nodig corrigeer.

  8. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

Let op!  Het muteren van lay-outs A t/m D is een 'Beveiligde functie', hiervoor is een wachtwoord nodig.
