Nieuw machine aanmaken of kopiëren

Een goede machineadministratie is een belangrijke basis voor veel andere Powerall Connect modules. Om een nieuw machine toe te voegen, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud machines (MBB).

  2. Kies voor de knop  Opvoer.

    • Een nieuw servicenummer wordt ingevuld en kan eventueel aangepast worden.

  3. Kies voor de knop  Enter.

    • Het scherm Nieuwe machine verschijnt.

  4. Vink eventueel een andere optie aan: Andere machine / Online (RDW) / Kentekenkaart / Handmatig.

    1. Optie Andere machine:

      1. Als u een machine wil kopiëren, geef dan het betreffend servicenummer in.

      2. Vink aan, wat overgenomen moet worden.

      3. Kies voor de knop  Doorgaan.

    2. Optie Online (RDW):

      1. Geef het kenteken in.

      2. Druk op de Enter-toets of kies voor de knop Opvragen.

      3. Bij de melding Wilt u de gegevens opvragen bij het RDW:

        • Kies voor de knop Ja.

          • Bij de melding De gegevens zijn succesvol opgehaald:

            Kies voor de knop  OK.

        • Controleer de gegevens (diverse gegevens worden overgenomen, zie ook FAQ).

      4. Geef overige gegevens in.

      5. Kies voor de knop  Doorgaan.

    3. Optie Kentekenkaart

      1. Geef de gegevens in.

        Opmerking: Hierbij is een kentekenkaartlezer nodig.

      2. Kies voor de knop  Doorgaan.

    4. Optie Handmatig

      1. Geef de gegevens in.


        Ingave hoofdgroep e.d. en BTW-type.

      2. Kies voor de knop  Doorgaan.

  5. Vul de benodigde gegevens op de tabbladen in, zie ook de velduitleg onderaan.

    Belangrijk: Vergeet niet om de inkoopprijs op te geven.

  6. Bij optie machine administratie 1.0 (standaard) is dit:


    NB De velden in het rode blok zijn afhankelijk van het soort bedrijf.

  7. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

Website(s)

Hoe komt het dat de website vinkjes niet worden opgeslagen of uitgezet?

Het gaat om tabblad Website of Commercieel, om de websitevinkjes bij website.

Als bij Instellingen machines, veld Update website(s) bij wijziging status is ja, worden alleen de machines met inkoopstatus Voorraad en verkoopstatus Nvt op ja gezet, in de overige gevallen worden de vinkjes uitgezet.
