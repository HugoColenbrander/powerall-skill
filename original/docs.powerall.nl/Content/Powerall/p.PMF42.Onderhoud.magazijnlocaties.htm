Onderhoud magazijnlocaties

Om een magazijn locatie toe te voegen of te wijzigen, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud magazijnlocaties (ABZ) .

    • In Menu 3.0 onder Artikelen > Overige > Configuratie.

  2. Kies voor de knop  Wijzigen.   of   Kies voor de knop Opvoer.

  3. Vanaf versie 3.20, kunnen deze naast Artikelen ook voor Machines gebruikt worden.

    Er kan hierbij ook een locatie omschrijving ingegeven worden.

  4. Geef de <locatiecode> in.

  5. Geef de pickzone en Putaway in.

  6. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

 

FAQ

Hieronder een uitleg van de velden van scherm Onderhoud magazijnlocaties:

 

Magazijncode De code van het magazijn.
 
Locatiecode De locatie code van de magazijnlocatie.
Omschrijving De 'betreffende' omschrijving.
 
Pickzone De zone waaruit gepickt wordt.
Putaway Put away is de plaats waar het product zich bevindt.

 

Opmerking: D.m.v. de knop Print label kan de er een label (m.b.v. layout M) van de magazijnlocatie(s) worden afgedrukt.
• Bij meerdere magazijnen kan een magazijn ingegeven worden.

Waar worden deze gegevens gebruikt?

Deze magazijn locatiecodes gegevens worden gebruikt voor:

  • Overzicht orders (te picken) 2.0

  • Print artikellabels

Wat betekent putaway?

Een goede magazijnuitvoering zorgt ervoor dat producten automatisch naar de juiste bestemmingslocatie gaan.

  • Putaway is het proces waarbij producten uit de ontvangende zending worden gehaald en op de meest geschikte plaats neergezet worden.

  • In Powerall Connect wordt ook de term inruimen gebruikt voor Putaway . Het is de (locatie) code waar je iets neerlegt.

    • Waarbij bijv. Putaway code A dan een 'snelloper' kan zijn en code B weer niet.
