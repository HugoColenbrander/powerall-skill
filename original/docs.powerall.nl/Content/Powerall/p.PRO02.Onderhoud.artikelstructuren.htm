Onderhoud artikelstructuren

De artikelstructuren worden gebruikt om bepaalde standaard artikelen samen te stellen. Deze kunnen gebruikt worden voor de machineverkoop, productie of projecten, samenstellingen, assemblies, assemblages of keuze artikelen.

 

Voorwaarden

Het artikel waaraan de structuur-artikelen gekoppeld zijn, moet aanwezig zijn evenals de artikelen in de regels.

 

Onderhoud

Om deze toe te voegen of te wijzigen, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud artikelstructuren (ASO).

  2. Geef de Structuur-artikelcode in.

  3. Kies voor de knop  Wijzigen.   of   Kies voor de knop Opvoer.

  4. Selecteer de soort structuur.

  5. Geef in de regels, de artikelen in.

    • Voor meer informatie, klik op velduitleg hieronder.

  6. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

Kan ik direct een werkorder of project starten?

Ja, dit is mogelijk. Om dit te doen, doorloopt u de volgende stappen:

  1. Kies voor de knop  Start werkorder.

    • Het scherm Start werkorder verschijnt.

    1. Geef de relatie in.

    2. Geef de overige gegevens zoals werkordersoort / geleverd vullen in.

    3. Kies voor de knop Opslaan.
  2. Kies voor de knop  Start project. (alleen bij productie 1.0)
Waar kan ik geen artikelstructuur code ingeven?

Samenstellingen en artikellijsten worden niet ondersteund bij:

  • Kassa
  • Contracten
  • Verhuur

  3. Geef de overige order gegevens in.

Hoe ziet de artikelstructuur eruit?

  1. Optie A keuzelijst is nee:

    • Opmerking: Voordeel keuze is verplicht geen kans op fouten, er moet er een keuze gemaakt worden, nadeel keuze maken kost iets meer tijd

  2. Optie B keuzelijst is ja:

    Belangrijk: Het aantal artikelcodes is hierbij verschillend, is meer een 'platte keuze' structuur.

    • Opmerking: Voordeel snellere keuze, keuze is niet verplicht. Nadeel kans op mogelijke fouten.
