Machine inkooporder

Om inkoop van een machine in te boeken, doorloopt u de volgende stappen:

  1. Ga naar Selecteren inkooporders (IS).
    Of

    Machines inkooporders (IM) vanaf versie 3.23.

  2. Kies voor de knop  Nieuw.

  3. Zoek de leverancier op en enter door de velden heen.

    Opmerking: Geef de gewenste leverdatum, referentie en ordergegevens en inkoper op, ter herkenning van de order in de historie.
    • Met de toets ‘Page Down’ toets ga je direct naar de 1e regel.

  4. Vul bij artikelcode bijv. VK-Mach in.

    • Het scherm Inkoop transactie verschijnt (bij een gesloten machine-administratie).

    • Bij machine met accessoires verschijnt het scherm Inkoopregel tbv machine:

  5. Instelling Gesloten machine-administratie:

    1. Het scherm Inkoop transactie verschijnt.

      Bij servicenummer selecteer de machine of zoek deze op d.m.v. F4.

      Opmerking: Om een bestaande (tweedehandse) machine in te kopen moet de status op dat moment zijn: Inkoop Nvt en verkoop nvt.


      Tip: Bij servicenummer kan ook direct het kenteken of chassisnummer opgegeven worden.

      Indien de machine niet onder de klant staat, kan een nieuwe aangemaakt worden.

      Klik hier voor meer informatie over het aanmaken van een nieuwe machine:

      Nieuwe machine aanmaken:

      1. Kies voor de knop  Nieuw. De nieuwe machine krijgt vanzelf een servicenummer (uniek nummer te vergelijken met unitnummer).

        • Het scherm Nieuwe machine verschijnt.
      2. D.m.v. ingave kenteken bij optie Online-koppeling (RDW) of kopie functie (gegevens overnemen van ...). Hierbij kan bijv. een ‘Demo/prijslijst’ machine geselecteerd worden, als die aangemaakt is.
      3. Vul de gewenste velden.
      1. Zet op tabblad Financieel:
        1. de inkoopstatus op nvt.
      2. Vul de inkoopprijs in (deze prijs verschijnt dan bij de bruto inkoop).
      3. Kies voor de knop Opslaan.

      De machine is aangemaakt.

    2. Controleer de bruto / inkoopprijs en eventueel BPM.

  6. Instelling Open machine administratie (i.c.m. machine administratie 2.0)

    • Selecteer in de kolom Machine de drie puntjes.

      • Geef de machine in

  7. Kies voor de knop  Akkoord.

    • De machine is toegevoegd op de inkooporder.

    • De verkoopprijs wordt overgenomen in de regel en de inkoopprijs van de machine is bijgewerkt.

  8. Kies voor de knop  Einde order.

    • Selecteren inkooporders (IS) verschijnt weer.

De inkooporder is toegevoegd.

Opmerking: Om de inkooporder af te drukken, selecteer de order(regel) en op druk de knop Bestelbon.

Artikelen op machinevoorraad boeken

Een machine wordt vaak samengesteld uit meerdere losse onderdelen, die bĳ verschillende leveranciers ingekocht worden.

  • Inkooporders worden vaak buitenom het systeem gemaakt of via een aflever-klaarmaak-werkorder doorgekoppeld naar een inkooporder. Hierdoor lĳken de aflever-klaarmaakkosten onterecht hoog uit te vallen en worden er veel interne omzet en kosten gegenereerd.

Het is mogelĳk om artikelen (machinedeel) direct in te kopen op een servicenummer als er met de machine subadministratie gewerkt wordt. Het volgende geldt hierbij:

  • In de kolom Machine kan een machine geselecteerd worden.

    1. Klik in de kolom Machine op de drie puntjes:

      • Het scherm Inkoopregel tbv machine verschijnt.

    2. Selecteer de machine.

    3. Kies voor de knop  Sluiten.

  • Bij Boeken goederenontvangst (AVO) wordt het volgende geboekt:

    • De balanswaarde van de machine wordt verhoogt en
    • de artikelvoorraad wordt bĳ ontvangen van de accessoires overgeboekt naar de machine.
  • De inkooporder kan op artikelniveau gematcht worden met de factuur.

  • Opmerking: De inkopen van de verschillende accessoires zĳn terug te vinden bij tabblad Transacties bĳ de machine.

Bij de Instellingen machines Gebruik machine administratie 2.0 is ja verschijnt het scherm Inkoopregel tbv machine. Bij het veld machine inkoop regeltype bestaat naast de optie Machinedeel (standaard) ook de opties:

  • Accessoire instelling
  • Machinedeel
  • Overige optioneel

In de regel verschijnt in de kolom type de betreffende regeltype:

Na bestelling/ontvangst en het doorverwerken van de machine transacties staat het bedrag vermeld, bij Onderhoud machines (MBB), tabblad Transacties onder Accessoire, bij bron staat:

  • Bestelling machinedeel

  • Ontvangst machinedeel

Bij Opvragen artikelen, tabblad Mutaties wordt het type AfgM. en Ink. geboekt:

BPM

Bij een inkooporder kan ook een artikel opgegeven worden, dat nodig is voor een machine. In de kolom Machine kan de machine geselecteerd worden met het inkoop regeltype. BPM kan hier geselecteerd worden.

Overige

Wat moet de status zijn van de machine bij inkoop / een inkooporder?

Bij Onderhoud machines, tabblad Financieel, moet het volgende status staan:

  1. De Inkoopstatus moet nvt of besteld zijn.

  2. De Verkoopstatus moet nvt zijn.

Is het mogelijk om bij een Machine-inkoop factuur de kostprijs te boeken?

Ja, dit is mogelijk i.c.m. als de koppeling grootboek uit staat bij de Instellingen artikelen en alleen bij een niet gesloten machine administratie.

Bij een verkoopfactuur wordt de omzet en kostprijs geboekt en bij een inkoopfactuur alleen de kostprijs.

Klik hier voor meer informatie over inkoop machine i.c.m. machine administratie 2.0

Ik wil na het ingeven van de machine met kostensoort Machine, dat het systeem vraagt of de verkoopkant van de machine leeg gemaakt moet worden.

  • Bij Ja, dan wordt 3XXX (Voorraadrekening) gebruikt.

  • Bij Nee, dan wordt de 7XXX (Kostprijsrekening) gebruikt.

Zodat de grootboekrekening aansluit bij de status van de machine.
