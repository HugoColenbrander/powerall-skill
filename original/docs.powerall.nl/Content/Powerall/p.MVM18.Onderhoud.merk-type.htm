Onderhoud merk/type (machine)

Om een type aan een merk toe te voegen, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud merk/type (MBT).

    • In Menu 3.0 onder Machine > Overige > Configuratie.

  2. Geef de merkcode in.

    • Bij F4 verschijnt het scherm Zoek machine merken.

  3. Kies voor de knop  Wijzigen.   of   Kies voor de knop Opvoer.

  4. Geef de machine type omschrijving in.

  5. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

 

FAQ

Hoe wijzig ik het machine merk/type?

Een machine type (merk) kan niet direct gewijzigd worden. Om het type te wijzigen, doorloopt u de volgende stappen:

  1. Verwijder het machine type.
  2. Voeg het juiste machine type weer toe d.m.v. de Opvoer knop.

Het merk/type is gewijzigd.
