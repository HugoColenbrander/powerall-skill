Overzicht voorraadmutaties

Bij alle voorraad 'handelingen' vinden er artikelmutaties plaats. Om deze op te vragen, doorloopt u de volgende stappen:

  1. Ga naar Mutatie overzicht (ALM).

    • Het scherm Overzicht voorraadmutaties verschijnt.

  2. Maak een selectie:

    • Klik hier voor meer informatie over de mutatiesoort(en) / type:

      1. Alle

      2. Logistiek komt alleen voor bij Overzicht voorraadmutaties.

        • Alleen de logistieke en + en - mutaties worden hierbij weergegeven, zonder prijswijzigingen, herwaarderingen e.d.

          Tip: Deze optie kan gebruikt voor controle (door de accountant) om de artikelmutaties te controleren.

      3. Afgifte

      4. Correctie - Artikel voorraad corrigeren

      5. Ontvangst - Boeken goederenontvangst (AVO)

      6. Inkoop - idem

      7. Verkoop

      8. Verwijzing - Artikel verwijzing aanleggen

      9. Multimagazijn - Multimagazijn overboeken

      10. Herwaardering, Periodieke voorraadaansluiting / artikelprijsmutaties

      11. Prijswijziging

      12. Machine accessoire / Afgm. zie Machine inkooporder

    • Bij mutatiesoort Multimagazijn is een selectie mogelijk op van en t/m magazijn:

      • Magazijncode levering

      • Magazijncode ontvangst

      Bij deze mutatiesoort wordt de locatie van het 'naar magazijn' getoond.

    • Het is mogelijk te sorteren op:

      • Artikelcode (standaard)

      • Locatie

      • Relatiecode, layout (B)

      • (Artikel) groep, layout (C)

    • Layout D Inruiminglijst / Eigen voorraad.

      • Als bij Instellingen artikelen tabblad Koppelingen de instelling Print verslag na goederenontvangst Eigen voorraad is, wordt layout D gebruikt. Dit geldt voor bijv.Boeken goederenontvangst.

      • BTB orders worden uitgesloten omdat deze vaak direct op de verkooporder of werkorder afgedrukt worden.

      • Inruimen: Goederen worden vanuit de ontvangstlocatie in het magazijn op voorraad gelegd.

  3. Matching

    • Selecteer de matching status.
      • Klik hier voor meer informatie over de matching status:

        1. Alle
        2. Ontvangen
        3. Proforma
        4. Gematcht
        5. Vervallen

  4. Geef eventueel de matching relatiecode / factuurnummer in.

  5. Kies voor de knop Afdrukken.

Het overzicht verschijnt op het scherm.
