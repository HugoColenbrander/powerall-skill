Project rapportages

Belangrijk: Onderstaand geldt voor projecten 2.0.

Van de projectgegevens is het mogelijk om een diverse overzichten met selecties aan te vragen. Om dit te doen, doorloopt u de volgende stappen:

  1. Ga naar Overzicht projecten (PL).

  2. Selecteer het overzicht.

    1. Projectstructuur (PLP) : Overzicht van de projectstructuur met niveaus, kostprijzen/afmetingen e.d.
    2. Projectcalculatie (PLC) : Overzicht project voorcalculatie / nacalculatie met beperkte gegevens incl. start - en einddatum, bedrag e.d.
    3. Projectpicklist (PLL) : Overzicht project picklijst.

      Opmerking: De selectie Alle regels printen is instelbaar bij Instellingen projecten, tabblad Productie.

  3. Geef een selectie op.

  4. Kies voor de knop Afdrukken.

Het overzicht verschijnt op het scherm.
