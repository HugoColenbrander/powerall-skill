Verhuur planbord

Het verhuur planbord geeft een goed overzicht van de verhuurde machines. Om naar dit planbord te gaan, doorloopt u de volgende stappen:

  1. Ga naar Verhuur planbord (HP).

    • Het scherm Verhuur planbord verschijnt.

  2. Het volgende is mogelijk:

    1. Er kan een selectie gemaakt worden op machines.

      1. Druk op de knoppen Lopende orders, Beschikbaar of Alle machines (standaard).
      2. Dubbel klik links op de machine, om de machine details te bekijken.
        • Het scherm Machine details verschijnt.
      3. Machine groepsselectie.

    2. Een order kan bekeken of gewijzigd worden:

      1. Selecteer in het planbord een machine, en dubbelklik op deze regel.

    3. De weergave kan gewijzigd worden:

      1. Met de knoppen Vandaag, Dag, 3 Dagen, Week en Maand kan snel wisselen worden tussen verschillende periodes.

      2. Ook kan met de blauwe schuifbalk de weergave vergroot of verkleind worden.

        Opmerking: De standaard periode weergave kan via de Instellingen verhuur/contract, tabblad Beheer, aangegeven worden bij 'Standaard weergave periode'.

  3. Sluit het planbord.

De kleuren geven de verhuurstatus weer.

 

FAQ

Komen afgemelde machines direct weer terug in het planbord?

Bij afmelden stopt de verhuurperiode.

  • Bij bijvoorbeeld een grotere machine die door het verhuurbedrijf opgehaald moet worden, met de vrachtwagen, kan het met de rittenplanning zo zijn, dat deze pas twee dagen later opgehaald wordt. De machine is dan dus pas twee dagen later retour en ook dan pas weer beschikbaar voor verhuur.

  • Er kan ook geconstateerd worden dat er iets kapot is of dat eerst de machine schoongemaakt en/of gecontroleerd moet worden.

Als de machine retour gemeld wordt, verschijnt de machine weer op het planbord.
