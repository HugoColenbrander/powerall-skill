Powerall login

Om in te loggen in Powerall Connect, doorloopt u de volgende stappen:

  1. Start op het bureaublad Powerall op.

    • Het inlog scherm verschijnt.

  2. De volgende talen zijn mogelijk instelling:

    1. Nederlands (standaard)

    2. English / Engels

    3. Français / Frans

    4. Deutsch / Duits

  3. Geef de gebruikersnaam en wachtwoord in.

    • Vink de gebruikersnaam onthouden aan.

  4. Kies voor de knop  Inloggen.

  • Het is mogelijk om in een andere taal in te loggen instelling.
    • Bij het Duits wordt de datum opgemaakt met punten 01.06.26 i.p.v. de standaard schuinestreep 01/06/26.

Belangrijk: Neem contact op met de planning om dit in te richten.
