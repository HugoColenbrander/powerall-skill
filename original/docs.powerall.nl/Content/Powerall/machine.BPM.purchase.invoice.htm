BPM inkoopfactuur en kentekenkosten

Bij een nieuwe personenauto's en motorrijwielen is BPM verplicht. Om de deze inkoopfactuur en kenteken kosten etc. te boeken, doorloopt u de volgende stappen:

  1. Ga naar Boeken inkoopfacturen (KI) of Inlezen e-factuur (KEI).

  2. Kies voor de knop  Nieuw boekstuk.

  3. Zoek de klant op, vul de overige gegevens in en enter door de velden heen.

  4. In de regels bij grootboek vul het volgende in:

    • bij nieuwe machines: Voorraad machines BPM.

    • bij gebruikte machines Voorraad machines BPM gebruikt.

    Dit betreft alleen de BPM.

    Opmerking: Het bedrag van de machine exclusief BTW wordt geboekt op de grootboekrekening Tussenrekening inkoop machines of natuurlijk automatisch als er een inkooporder van de machine is gemaakt.

  5. Het scherm Kostendrager verschijnt.

    1. Bij Servicenummer kies je de machine. De machine informatie met het vinkje bij BPM factuur verschijnt.

    2. Kies voor de knop Opslaan.

  6. Vervolgens kunnen er extra kosten, zoals afleverkosten, kentekenkosten etc ingeboekt worden. Vul hiervoor de grootboekrekening in van Tussen rekening ontvangen machines en enter verder,

    1. Vervolgens kan de machine worden geselecteerd. De machine informatie verschijnt weer.

    2. Kies voor de knop Opslaan.

    Bij kolom kostendrager, komt het machine- of servicenummer te staan, je kan dit zo overtypen.

    Tip: Handig is om het servicenummer te kopiëren.

  7. Kies voor de knop  Einde boekstuk.

De boeking is verwerkt.

 

Ontvangen machine

Belangrijk: Een machine ontvangst mag niet via Machine ontvangst / levering (MVO) ontvangen worden, als deze op een inkooporder staat.

Als er nog facturen ontvangen worden, doorloopt u de volgende stappen:

  1. Ga naar Machine Ontvangst/levering (MVO).

    • Ontvangst is al aangevinkt.

  2. Plak of typ daar het servicenummer in van de vorige stap.

  3. Vul bij bedrag (ontvangst) het totale kostenbedrag in bijv. afleverkosten, kentekenkosten etc.

    Opmerking: Als de facturen apart ontvangen worden, wordt de ontvangst ook apart geboekt.

  4. Kies voor de knop Opslaan.

De kosten zijn op de machine geboekt.

Opmerking: Bij Opvragen machines of via F8 zijn bij tabblad Transacties deze gegevens te zien.
