Onderhoud accessoires verhuur 2.0

Met accessoires kunnen bepaalde artikelen mede verhuurd worden. Een accessoires is altijd gekoppeld aan een verhuurtarief groep

Om een accessoire toe te voegen of te wijzigen, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud accessoires (HOA).

    • In Menu 3.0 onder Verhuur > Contracten > Overige > Configuratie.

  2. Geef het artikel in.

    • Bij F4 verschijnt het scherm Zoek artikelen.

  3. Kies voor de knop  Wijzigen.   of   Kies voor de knop Opvoer.

  4. Indien nodig, pas de gegevens aan.

  5. Geef de mogelijke serienummers, magazijn en optionele aantallen in.

  6. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

Instellingen

De accessoires moeten aan een verhuurgroep accessoires gekoppeld worden bij:

  • Onderhoud verhuurtarieven 2.0 vink hiervoor Accessoires onderaan aan.

De machine hoofdgroep accessoires moet bij Onderhoud machine (hoofd- en sub) groepenhet veld Machinegroep type moet op accessoires staan.

Magazijngroep type Optie voor magazijngroep type m.b.t. de verhuur bij Zoeken machine groepen.
- Machine (standaard)
- Accessoire, alleen zichtbaar bij verhuur accessoires.
Wat gebeurt er bij accessoires i.c.m. de voorraad?

Het volgende gebeurt met de accessoires voorraad bij:

  • Afgifte naar beneden / verminderd.

  • Retour weer omhoog / vermeerderd.
