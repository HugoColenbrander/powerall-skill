SMS / WhatsApp versturen werkgereed

Als een werkorder gereed is, kan er een SMS of WhatsApp verstuurd worden, dat de machine opgehaald kan worden. Om dit te doen, doorloopt u de volgende stappen:

  1. Ga naar Selecteren werkorders (WS).

  2. Selecteer de werkorder.

  3. Kies voor de knop  < Meer functies.

    1. Kies voor de knop  Verstuur SMS of Verstuur WhatsApp.

    2. Kies voor de knop Versturen.

Het bericht is verzonden.

Opmerking: Bij Opvragen relaties, tabblad CRM, is het verzonden bericht zichtbaar. Er kan een standaard tekst ingevuld worden.
