Machine kaart verzenden

Het is mogelijk om een machinekaart, met bepaalde gegevens, te verzenden per e-mail. Om dit te doen, doorloopt u de volgende stappen:

  1. Ga naar Machines.

  2. Selecteer een machine.

  3. Android: Kies voor 3 puntjes knop, rechtsboven. > Machine sturen.
    iOS: Kies voor de mail knop.

    • Als de nieuwe Machine 'kaart' instellingen nog niet zijn ingesteld, verschijnt er een pop-up melding: Kan machine niet verzenden, vanaf versie 3.29.

  4. Vul het <E-mailadres> in en de overige gegevens.

  5. Kies voor de knop VERSTUREN.

  6. Vanaf versie 3.29 kan de machinekaart opgemaakt worden in een eigen stijl.

    • Tevens kan er een aanbiedingsprijs bij Uw prijs worden ingegeven.

      Deze prijs is eenmalig d.w.z. wordt niet opgeslagen.

Het PDF document met de machinegegevens is verstuurd.

 

FAQ

Hoe ziet de e-mail eruit die de klant ontvangt?

Hierbij een voorbeeld, incl. foto's, hoe een e-mail eruit kan zien, vanaf versie 3.29.

Zie Layout bij machine label, voor een dit voorbeeld.

Waar kan ik zien dat er een machinekaart verzonden is, vanaf versie 3.29?

In de API log is te zien, dat een machine mail verzonden is, zie:

  • Connect API logging

  • In de kolom Bericht staat [Label verstuurd naar <e-mail> voor machine <X>].

  • of bij een mislukt bericht: [Geen label verstuur voor machine <xx>, layoutcode is niet ingevuld].

Wat moet er ingesteld zijn, vanaf versie 3.29?

Belangrijk: Neem contact op met de planning om dit in te richten.

Het volgende moet hiervoor ingesteld zijn:

  1. Er moet een Layout code voor de machine aangemaakt worden m.b.t. programma MVM27. Deze layout-code wordt gebruikt voor het PDF bestand met de machinegegevens, als e-mail bijlage.

    • In deze layout bijv. K kan een nieuw veld SYS440, gebruikt worden, hiermee kan de prijs zoals ingegeven op de app weergegeven worden op de machinekaart offerte.

    • In 3.31 zijn de volgende velden nog toegevoegd:

      • BTW soort / type (Nieuw / Gebruikt)

      • Personeelsnaam

      • Functie

      • Mobielnummer

  2. Bij Instellingen machines, tabblad Printvoorkeuren, moet bij machine label e-mailen deze Layout ingevuld worden.

    • Layoutcode
      - Machine label
      - Machine basislijst
      - Machine voorraadlijst
      - Machine ontvangst
      - Machine levering
      - Machine label e-mailen      		 De betreffend Layout codes voor:
      - Leeg (standaard)
      - Ontvangst, standaard: O
      - Levering, standaard: L
      - Machine label e-mailen, leeg (standaard), zie Machine kaart verzenden vanaf versie 3.29

  3. Bij Instellingen documentbeheer, moet tabblad Machines ingevuld zijn, bijv.:

    In het bodytekst bestand kunnen 2 tags gebruikt worden:

    1. <crm_msg> Het bericht wat in de CRM app ingegeven kan worden.

      • Het standaard bericht is: In de bijlage vindt u de machinekaart.

    2. <crm_nam> De (afzender) omschrijving van de ingelogde CRM app gebruiker.

      Hierbij een voorbeeld van een body bestand.

    Er wordt een cc mail gestuurd, bij het gebruik van een standaard bcc e-mail adres.

  4. Bij de betreffende personeelscode d.w.z. de CRM app gebruiker moet een e-mailadres ingegeven zijn, anders wordt het algemene e-mailadres van bovengenoemde instelling gebruikt. Dit e-mail adres wordt als afzender gebruikt.

  5. De CRM app gebruiker moet altijd gekoppeld zijn, aan een personeelscode.

Als de CRM app nog niet is geüpdatet naar versie 3.29, wordt de oude verzendwijze gebruikt.

 

Als bovengenoemde niet ingesteld is, verschijnt de melding:

  • Kan machine niet verzenden Sluiten

  • Stel bovenstaande in om de machine info te versturen.
