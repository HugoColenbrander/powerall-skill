Onderhoud magazijnkratten

Opmerking: Alleen te gebruiken bij de optie Multi-magazijnen.

Bij een magazijnkrat kan de kleur opgegeven worden en een label worden afgedrukt. Ook is er info over het huidige gebruik. Om deze toe te voegen of te wijzigen, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud magazijnkratten (AM?) .

    • Het scherm Onderhoud magazijnkratten verschijnt.

  2. Kies voor de knop  Wijzigen.   of   Kies voor de knop Opvoer.

  3. Geef het<kratnummer> in.

    Belangrijk: Geef het kratnummer een uniek nummer, d.w.z. geen nummer dat als artikelcode bestaat.

  4. Selecteer de kleur.

  5. Geef het scan ID in

  6. Kies voor de knop  Print. optioneel

    • Voor het afdrukken van een kratlabel.

    • Voor de krat label layout wordt PMF27 gebruikt, rapport 2 is krat gegevens.

  7. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

 

FAQ

Opmerking: Het standaard magazijn is het default pickwarehouse magazijn.

Hieronder een uitleg van de velden van scherm Onderhoud magazijnkratten:

 

Kratnummer Het nummer van de krat.
Kleur Optie voor de kleur van de krat:
- Rood (standaard)
- Oranje / Geel / Groen / Lichtblauw / Donkerblauw / Paars / Zwart / Wit / Bruin
Scan ID Het scan id van de krat, gebruikt voor de barcodescanner.
 
Info huidig gebruik  
Taaknummer Het nummer van de taak.
Ordernummer Het huidig ordernummer, waar de krat momenteel aan gekoppeld is.
 
Magazijn Het magazijn(nummer), zie Onderhoud magazijnen.
- 0 Allen (standaard).
Locatie De fysiek of standaard locatie van de krat.

 

Waar kan ik kratten gebruiken?

De kratten kunnen gebruikt of gescand worden i.c.m. de barcodescanner.

  • Zie Barcode scanner.

  • Bij Overzicht orders (te picken) 2.0 tabblad Gepickt kan bij het veld Direct zoeken picklist/ID het ScanID van de krat gescand / ingegeven worden.

  • Op Werkbon app icm accepteren kratten, vanaf versie 3.30.

    Dit kan zonder bovengenoemde kratnummers.
