Nieuwe machine CRM app

Opmerking: Alleen beschikbaar bij autorisatie.

Om een nieuwe machine toe te voegen op de Powerall CRM app, doorloopt u de volgende stappen:

  1. Ga naar het Dashboard.

  2. Ga naar Relaties CRM app.

    1. Selecteer de relatie.

    2. Ga naar tabblad Machines.

  3. Kies voor de knop  (PLUS)

    • Het scherm Nieuwe Machine / Machine aanmaken verschijnt.

    • iOS  

  4. Vul machinegegevens i of zoek deze op.

    • Selecteer het juiste merk, hoofd - en subgroep.

    • Voer het serienummer in, let op zonder vreemde tekens/spaties etc.

    • Verkocht = eigendom van klant.

    • Configuratie vullen zover als relevant.

  5. iOS: Kies voor de knop Opslaan.

    Android: Kies voor de knop  MACHINE AANMAKEN.

De machine is aangemaakt, bij de relatie.

 

FAQ

Opmerking: Het merk en type zijn verplicht.

Hoe kan ik het beste een nieuwe machine toevoegen?

  1. Vanuit Relaties CRM app > relatie > tabblad Machines.

  2. Wanneer het een machine betreft, die onderhouden/gerepareerd gaat worden.

  3. Wanneer het een machine betreft, die wellicht vervangen gaat worden, zodat je alle gewenste info als input voor de offerte kunt gebruiken.

Hoe komt het dat ik de plus knop niet zie?

Het is mogelijk om het toevoegen van een nieuwe machine te autoriseren.

  • Controleer bij Onderhoud personeelscodes (BTP), tabblad CRM app de autorisatie bij het veld machines.
