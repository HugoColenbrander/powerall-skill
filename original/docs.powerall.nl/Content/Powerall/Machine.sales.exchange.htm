Machine verkoop met inruil

Om een machine te verkopen met een inruil, doorloopt u de volgende stappen:

  1. Ga naar Selecteren verkooporders (VS). (of Machine orders)

  2. Kies voor de knop  Nieuw.

  3. Zoek de klant op en enter door de velden heen.

    Opmerking: Geef de afleverdatum, referentie en ordergegevens op, ter herkenning van de order in de historie.
    • Met de toets Page Down toets ga je direct door naar de regels.

  4. Vul bij artikelcode bijv. VK-Mach in.

    • Het scherm Verkoop transactie verschijnt.

  5. Vink Inruil aan.

  6. Geef de machine in.

    Of Druk op het vergrootglas en selecteer de machine.

    Het volgende is mogelijk:

    • Automatisch worden de machines getoond van de klant.

    • Indien de machine niet onder de klant staat, kan een nieuwe aangemaakt worden.

      • Nieuwe machine aanmaken:

      1. Kies voor de knop  Nieuw. De nieuwe machine krijgt vanzelf een servicenummer (uniek nummer te vergelijken met unitnummer).

        • Het scherm Nieuwe machine verschijnt.
      2. D.m.v. ingave kenteken bij optie Online-koppeling (RDW) of kopie functie (gegevens overnemen van ...). Hierbij kan bijv. een ‘Demo/prijslijst’ machine geselecteerd worden, als die aangemaakt is.
      3. Vul de gewenste velden.
      1. Vul op tabblad Financieel het volgende in:
        1. de Inkoopstatus op nvt.
        2. de Verkoopstatus op Verkocht / Eigendom.
        3. de Eigenaar de huidige eigenaar in, die de machine inruilt.
        4. de inkoopprijs, deze verschijnt dan bij bruto inkoop, of vul deze later in.
      2. Kies voor de knop Opslaan.

      De machine is aangemaakt.

  7. Controleer BTW-type / artikelgroep (deze wordt overgenomen van de machine):

    • Marge, indien inruil zonder BTW.

    • Gebruikt, indien inruil met BTW.

  8. Vul de bruto inkoopprijs in.

  9. Kies voor de knop  Akkoord.

  10. De machine wordt toegevoegd op de verkooporder.

De verkooporder kan verder worden afgewerkt zie Machine verkoop.

 

Ontvangen inruil machine (gesloten administratie)

Belangrijk: Een machine ontvangst mag niet via Machine ontvangst / levering (MVO) ontvangen worden, als deze op een inkooporder staat.

Een machine die ingeruild wordt, moet net als een nieuwe machine ontvangen worden. Om dit te doen, doorloopt u de volgende stappen:

  1. Ga naar Ontvangst / levering (MVO).

  2. Vink Ontvangst aan.

  3. Druk vervolgens op het vergrootglas en de machines worden getoond, die besteld zijn of op inruil wachten.

  4. Selecteer de te ontvangen machine.

  5. Kies voor de knop Opslaan.

De machine is ontvangen en komt op voorraad te staan.

  • De inkoopstatus is dan Voorraad en verkoopstatus Nvt.

 

FAQ

Wat moet de status van de machine bij inruil zijn?

Bij Onderhoud machines, tabblad Financieel, moet het volgende staan:

  1. De 'Inkoopstatus' moet nvt of consignatie zijn.

  2. De 'Verkoopstatus' moet Verkocht / Eigendom zijn.

Wat gebeurt er bij een inruil van een machine?

De machine gegevens worden als volgt bijgewerkt:

  • op tabblad Service wordt de eindgebruiker / locatie leeg gemaakt. Dit om mogelijke problemen met servicemelding na verkoop van de machine te voorkomen.

  • op tabblad Financieel

    • Het veld eigenaar wordt gewist.

    • Open machine administratie:

      • De inkoopstatus wordt Voorraad en verkoopstatus wordt Nvt.

    • Gesloten machine administratie met subadministratie machines:

      • De inkoopstatus wordt Inruil (verwacht) en verkoopstatus wordt Verkocht / Eigendom, totdat de ontvangst geboekt wordt.

Belangrijk: Als een nieuwe (aanwezige) machine na bijv. 5 jaar ingeruild wordt, controleer dan het BTW-type, deze moet niet nieuw zijn maar bijv. gebruikt.
