Machine rapportages

In menu 'Overzichten machines'  kunnen diverse overzichten zoals, machine voorraad(waarde)lijst, machine groepen, basisgegevens, verkopen etc. aangevraagd worden.

Om een overzicht aan te vragen, doorloopt u de volgende stappen:

  1. Ga naar Overzichten machines (MP).

  2. Selecteer het overzicht.

    1. Machine voorraad (MPP): Overzicht van machines die op voorraad zijn. Met diverse selectie mogelijkheden, eventueel ook met foto (lay-out C).

      • Bij machine subadministratie en verhuur /contract machine vink Inclusief .. aan.

    2. Machine groepen (MPG): Overzicht van de machinegroepen.

      Klik hier, voor meer informatie over dit onderwerp.

      Bij soort lijst overzicht is er de volgende selectie mogelijkheid:

      • Eigen groepen (standaard)
      • Ingebouwde (advertentie) groepen
      • Alle groepen

      Opmerking: De Ingebouwde advertentiegroepen configuratie kan niet aangepast worden, wel gekopieerd.

    3. Machine basisgegevens (MPB): Alle machines met hun basisgegevens, inclusief configuratie.

    4. Machine verkopen (MPV): Overzicht van de verkochte machines, met de inkoop-, verkoopprijs en marge.

      • De inruil transacties worden niet weergegeven.

      • Volgens de instelling Basis verkoopmarge berekening wordt de marge berekend

      • Bij het gebruik van BPM kan er (met het BPM bedrag) rekening gehouden worden bij het berekenen van de marge, zoals bij F8 vanaf versie 3.28.

    5. Machinelabels (MPL): Hiermee worden machine labels afgedrukt.

    6. Machinepaspoort (MPS): Een overzicht de historie van de machines. Alle transacties zoals inkoop, inruil, verkoop, reparatie, verhuur, correcties worden bij dit 'paspoort' weergegeven.

    7. Reparatiehistorie (MPR): Gedetailleerd overzicht van de reparaties met de onderdelen per machine, inclusief tellerstand.

    8. Configuratie / machines (MPC): Overzicht van de machine a.h.v. de configuratie.

      • Selectie van één of meerdere configuratie elementen is hierbij mogelijk.

    9. Onderhoudsschema's (MPO): Overzicht van de onderhoudsschema's.

  3. Geef een selectie op.
  4. Kies voor de knop Afdrukken.

Het overzicht verschijnt op het scherm.

 

Afleverlijst machines (verkopen)

Om in een overzicht aan te vragen van de uit te leveren (verkochte) machines, doorloopt u de volgende stappen:

  1. Ga naar Overzichten verkooporders uitgebreid (VLO).

  2. Indien nodig maak een selectie.

  3. Bij Lijst soort, selecteer Afleverlijst machines.

  4. Kies voor de knop Afdrukken.

Op dit overzicht worden de machinegegevens met de klantnaam, merk, type, kleur, nieuw/gebruikt/marge en verkoper afgedrukt.
