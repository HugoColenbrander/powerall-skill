Windows gebruikersaccount melding

Als er een app of een programma geïnstalleerd wordt, op de server of computer, geeft Windows een (gebruikersaccountbeheer) melding:

  • Wilt u toestaan dat deze app wijzigingen aan uw apparaat aanbrengt?

Om de app te installeren, doorloopt u de volgende stappen:

  • Kies voor de knop Ja.

Het betreffend programma wordt geïnstalleerd.

Belangrijk: Administrator - of beheersrechten zijn vereist om op deze pc/(terminal)server/werkstation Powerall Connect of andere software te installeren of bij te werken.
