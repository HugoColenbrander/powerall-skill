Onderhoud standaard configuratie (machines)

Aan een machine kan een bepaalde configuratie worden toegewezen. Het volgende is mogelijk:

  • Op basis van deze configuratie kunnen bepaalde overzichten aangevraagd worden.

  • De configuratie worden weergegeven bij/op:

    • Bestelbon

    • Powerall CRM app Machines CRM app bij kenmerken

    • Werkorder

    • Word (machine) offerte

    • Export website(s)

Om een configuratie-element toe te voegen of te wijzigen, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud standaard configuratie (MCB).

    • In Menu 3.0 onder Machine > Overige > Configuratie.

  2. Geef de configuratiecode in.

    • Bij F4 verschijnt het scherm Zoek standaard configuratie.

  3. Kies voor de knop  Wijzigen.   of   Kies voor de knop Opvoer.

    1. Bij opvoeren, geef de configuratiecode in.

    2. Selecteer de te kopiëren configuratiecode of kies voor de knop Annuleren.

  4. Wijzig de gegevens of geef deze in.

  5. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

Waar wordt de machine configuratie allemaal gebruikt?

De machine configuratie wordt o.a. gebruikt bij:

  • Onderhoud machines en Opvragen machines gegevens.

  • Selecteren machines
    • Met selectie mogelijkheden op configuratie elementen
  • Keuringslijst
  • Servicepaspoort
  • Reparatiehistorie
  • Export machines
  • Machine basislijst en verkooplijst
  • Standaard productstructuren
  • Offerte / opdrachtbevestiging
Wat betekenen de kolommen WO, Off en etc?

In de kolommen WO, Off VO, IO en VH kan aangegeven of de elementen weergegeven worden:

  • WO = Werkorders
  • Off = Offerte
  • VO =  Verkooporders
  • IO = Inkooporders
  • VH = Verhuurorders

    • Bij Onderhoud configuratie elementen (machines) kan dit standaard per element aangevinkt worden.
