Werkorder verwijderen

Het verwijderen van een werkorder kan op 2 manieren:

  • Direct via de Werkorder module.
  • Vanuit een servicemelding via de Servicemeldingen module.

Let op!  Het verwijderen van een werkorder kan niet ongedaan gemaakt worden en de informatie van de werkorder is niet meer op te roepen.

Vanuit de werkorder module

  1. Ga naar Selecteren werkorders (WS).

  2. Selecteer de werkorder.

  3. Kies voor de knop  Wijzig.

    • Het scherm Invoeren/wijzigen werkorders verschijnt.

  4. Kies voor de knop  Verwijder order.

    • De volgende pop-up verschijnt:

    • Kies voor de knop Ja.

De werkorder is verwijderd.

 

Vanuit een servicemelding

  1. Selecteren servicemeldingen (SS).

  2. Selecteer de servicemelding van de werkorder.

  3. Kies voor de knop  Beheer.

  4. Selecteer de werkorder.

  5. Kies voor de knop  Wijzig.

    1. Zie verder hierboven punt 4.

    2. Kies voor de knop  Verwijder order.

De werkorder is verwijderd.

 

Verwijderen meerdere werkorders

Voor het verwijderen van meerdere werkorders, doorloopt u de volgende stappen:

  1. Ga naar Verwijderen werkorders (WVV).

  2. Geef een selectiecriteria in.

  3. Kies voor de knop  Verwijderen.

    • Bij print verslag 'Ja' verschijnt er een overzicht van de verwijderde gegevens.

De werkorders zijn verwijderd.

 

FAQ

Kan ik een werkorder verwijderen waarop uren geregistreerd staan?

Nee, dit is niet mogelijk, de melding verschijnt:

  • Verwijderen werkorder is niet mogelijk omdat er al uren geregistreerd zijn.
