Systeemeisen

Uw systeem moet voldoen aan de volgende hardware- en software eisen (standaard productinstellingen) voor een naadloze uitvoering van Powerall Connect:

Hardware

 

Werkstation

  • Minimaal Windows 11 en Windows Server 2016.

  • Resolutie:

    • minimale 1366x768.

    • maximale 2560x1440 of 4K (3840x2160) op 150%.

  • Bij urenklokken via het touchscreen is de minimale resolutie Full HD (1920x1080).

  • Een installatie van Office (inclusief Word, Excel en Outlook) is noodzakelijk voor gebruik van sommige functionaliteit.

  • De prestatie-eisen (CPU, geheugen) zijn gelijk aan de eisen van Microsoft Office.

Server

  • Minimaal Windows 11 en Windows Server 2016.

  • Prestatie-eisen zijn afhankelijk van de hoeveelheid gebruikers, functionaliteiten en overige software op het systeem.

  • Hierbij kunnen de volgende uitgangspunten gehanteerd worden:

    • Powerall Connect is zeer CPU-intensief. De effectieve snelheid van is vrijwel volledig afhankelijk van de snelheid van de CPU van de server (en in mindere mate het werkstation).

    • Een enkel Powerall Connect-proces (voor 1 gebruiker) gebruikt slechts 1 ‘thread’, de ‘single-core’ snelheid van de CPU is hierbij dus van groot belang.

    • Om meer gelijktijdige gebruikers aan te kunnen, helpt het uiteraard om beschikking te hebben over een groter aantal CPU-cores.

    • Powerall Connect gebruikt weinig geheugen, maximaal 100 MB per gebruiker. Er zijn enkele uitzonderingen, bijv. bij het importeren van grote hoeveelheden gegevens vanuit grote XML-bestanden.

    • Voor opslag raden we SSD’s aan, in verband met de veel hogere snelheid t.o.v. een klassieke harde schijf.

    • De benodigde opslagruimte is volledig afhankelijk van de grootte van de administratie en dossiers.

Netwerk

De verbinding tussen het werkstation en de server vereist een stabiel netwerk met een zeer lage latency (maximaal 1-2 ms). Dit werkt dus alleen in een lokaal netwerk en is niet bruikbaar over het internet (bijvoorbeeld via een VPN).

Om toch extern te kunnen werken, kunt u uiteraard gebruik maken van een terminal server waarop Powerall Connect geïnstalleerd is.

Een werkende internetverbinding is (hoewel niet strikt noodzakelijk) ten zeerste aan te raden. Dit is uiteraard wel vereist om digitaal te communiceren via koppelingen of e-mails te sturen. Ook ontvangt u via internet automatisch productupdates.

Software

Naast een bijgewerkte Windows-omgeving (minimaal Windows 11 en Windows Server 2016) vereist Powerall Connect .NET 4.6 en een aantal Visual C runtimes. Deze worden automatisch geïnstalleerd door de setup.
