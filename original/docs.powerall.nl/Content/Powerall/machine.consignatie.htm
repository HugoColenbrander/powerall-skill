Machine consignatie

Bij een consignatie verkoop komt de machine bij een bedrijf ter verkoop in de showroom te staan.

  1. Als de klant de machine brengt, wordt er een inkoopverklaring getekend. Deze moet ingescand worden.

  2. De inkoopverklaring wordt opgeslagen in een map in Powerall Connect bijv. het machine dossier.

    1. Ga naar Onderhoud machines (MBB).

    2. Ga naar tabblad Dossier.

    3. Kies voor de plus-knop .

      Selecteer het bestand en kies voor de knop Openen.


      Het bestand wordt toegevoegd aan het dossier.

  3. De machine wordt opgevoerd in Powerall Connect met inkoopstatus 'consignatie'.

    1. Ga naar Onderhoud machines (MBB).

    2. Ga naar tabblad Financieel.

    3. Kies voor de knop  Wijzig.

    4. Selecteer bij inkoopstatus Consignatie.

    5. Kies voor de knop Opslaan.

    De machine heeft de status consignatie.

Het is ook mogelijk om een nieuwe machine op te geven bij een inkoop- of verkooporder (transactie).

Opmerking: Bij 'Selecteren machines' zijn consignatie machines zichtbaar.

Verkoop

Als de machine verkocht wordt, doorloopt u de volgende stappen:

  1. Dat moet gebeuren door de inkoopverklaring in te boeken als inkoopfactuur.
    De status moet dan gewijzigd worden naar 'besteld' zodra als er een inkoopfactuur wordt geboekt op een machine met status 'consignatie'.

  2. Bij de ontvangst wijzigt deze dan automatisch naar 'voorraad', zoals gebruikelijk in de machine subadministratie.

  3. Daarnaast moet de inkoopfactuur betaald worden.

  4. Zodra de machine in een verkooporder/factuur/bon komt, moet deze eerst van status consignatie afgehaald worden.

Vervolgens moet de verkocht machine nog geboekt worden op de normale wijze.

 

Overzicht machinevoorraad

Het machine voorraad overzicht kan bij Machine voorraad (MPP) opgevraagd worden.

 

Normale administratie

  • Om machines met inkoopstatus consignatie te tonen, selecteer bij lijstsoort consignatie.

Gedeelde administratie

  • Bij een gedeelde administratie worden de machines met inkoopstatus consignatie ook getoond bij lijstsoort 2 Verkoop voorraad.
