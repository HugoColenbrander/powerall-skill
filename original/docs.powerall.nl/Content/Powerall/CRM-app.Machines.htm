Machines CRM app

Om naar de machines te gaan, doorloopt u de volgende stappen:

  1. Ga naar het menu of terug <.

  2. Kies voor de knop  Machines.

    • Het scherm Machines verschijnt.

  3. Geef bij Zoeken een zoekterm in.

    • De machines die voldoen aan de zoekterm worden weergegeven tijdens het intypen.

  4. De volgende tabbladen zijn aanwezig:

    • Algemeen

    • Afbeeldingen

      • Standaard

      • Website

      • Zie ook Machine foto's toevoegen

    • Transacties

      • De historie van de machine wordt weergeven. Druk op de order om de details te bekijken.

      • Voor meer info Machine transacties

Hoe kan ik de machines filteren?

Het is mogelijk de machines te filteren op:

  • Groepen

    1. Druk op Machines/voertuigen >

      • Een overzicht van de machine groepen wordt weergeven.

    2. Kies voor de knop  BEKIJK ALLES.

    3. of
      Selecteer een groep.
      • De eventuele subgroepen verschijnen

    4. De machines van de betreffende groepen verschijnen

  • Inkoop- / verkoopstatus.

Opmerking: Als er een gefilterd wordt verschijnt kort onderaan een melding hiervan:
- Er worden x filters toegepast op de inkoopstatus, verkoop

 

Op welke statussen kan ik de machines allemaal filteren?

De machines kunnen op de volgende status gefilterd worden:

  • Inkoopstatus

    • Alle

    • Nvt

    • Besteld

    • Inruil (verwacht)

    • Voorraad

    • Consignatie

  • Verkoopstatus

    • Alle

    • Nvt

    • Optie

    • Consignatie

    • Verkocht / Eigendom

    • Huur

    • Lease

Om dit te doen, doorloopt u de volgende stappen:

  1. Kies voor de knop (filteren).

  2. Vink de gewenste statussen aan.

  3. iOS: Kies voor de knop Klaar.

    Android: Kies voor de knop  TOEPASSEN.

De machines met deze status(sen) worden weergegeven.

Hoe kan in de machinekaart/gegevens (een PDF) verzenden?

Het is mogelijk om alle machinekaart te verzenden per e-mail. Om dit te doen, doorloopt u de volgende stappen:

  1. Ga naar Machines.

  2. Selecteer een machine.

  3. Android: Kies voor 3 puntjes knop, rechtsboven.
    iOS: Kies voor de mail knop.

  4. Druk op Machinekaart sturen.

  5. Vul het <E-mailadres> in en de overige gegevens.

  6. Kies voor de knop VERSTUREN.

Het PDF document met de machinegegevens is verstuurd.

Hoe wordt de handelscode gecodeerd of weergegeven bij machines?

De inkoopprijs van een machine wordt als handelscode weergegeven en zo gecodeerd dat de klant dit niet zomaar kan herkennen.

  • De handelscode wordt alleen weergegeven als de klantweergave uit staat.

  • Als de klantweergave aan staat wordt helemaal niks getoond (geen inkoopprijs / geen handelscode), alleen de adviesprijs.

De duizendtallen staan achteraan, de honderdtallen vooraan. Het derde karakter van de omschrijving wordt extra tussengevoegd, bijv.:

14500 wordt 50X14,
900 wordt 90X00 en
50 wordt 05X00.

Van prijzen boven honderdduizend wordt alleen het deel onder het honderdduizendtal omgezet bijv. 125.000 wordt dus 00X25.

Hoe wijzig ik een machine (configuratie)?

Om de machine (configuratie) te wijzigen, doorloopt u de volgende stappen:

  1. Ga naar Machines.

  2. Selecteer een machine.

  3. iOS: Kies voor de Pen knop.

    Android: Kies voor de knop  WIJZIGEN.

  4. Scrol naar beneden.

  5. Druk op de betreffende configuratie element > Maak een keuze.

  6. Kies voor de knop  WIJZIGINGEN OPSLAAN.

De gegevens zijn opgeslagen.
