Onderhoud garantiecodes

Aan een machine kan een bepaalde soort garantie gekoppeld worden. Om een garantiecode toe te voegen of te wijzigen, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud garantiecodes (MBA).

    • In Menu 3.0 onder Machine > Overige > Configuratie.

  2. Geef de garantiecode in.

    • Bij F4 verschijnt het scherm Zoek garantiecodes.

  3. Kies voor de knop  Wijzigen.   of   Kies voor de knop Opvoer.

  4. Wijzig de gegevens of geef deze in.

  5. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

 

FAQ

Hieronder een uitleg van de velden van scherm Onderhoud garantiecodes:

 

Garantiecode De garantiecode of garantieconditie, zie Onderhoud garantiecodes.
Afkorting De afkorting van deze code.
Omschrijving De 'betreffende' omschrijving.

 

 

Waar kan ik o.a. de garantie omschrijving zien?

Naast Onderhoud machines is de garantieomschrijving te zien:

  • Op het machine transactie scherm.

  • Bij Nieuwe servicemelding aanmaken.

Waar kan ik de garantiecode opgeven bij een Machine?

Het is mogelijk om de garantiecode bij Onderhoud machines (MBB), tabblad Service op te geven, bij het veld Garantie conditie.

 

Wat kan ik allemaal bij een machine garantie opgeven?

De volgende velden kunnen gebruikt worden:

Datum in gebruik De datum dat de machine in gebruik genomen is.
Einddatum garantie De einddatum van de garantie. Geeft een melding in de werkorder, als de machine nog in de garantie valt.
Garantie t/m urenstand De garantie t/m de urenstand.
Garantie conditie De garantie conditie zie Onderhoud garantiecodes.
