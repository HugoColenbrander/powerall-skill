Financiële analyse

Een echte must, is het maken van een financiële analyse. Hoe staat de organisatie er financieel voor is de hoofdvraag. Hiervoor wordt gebruik gemaakt van financiële kengetallen.

Liquiditeit

De liquiditeit laat zien of de organisatie op korte termijn aan haar schulden kan voldoen. Met andere woorden kan de organisatie de korte termijn schulden betalen?

Current ratio

Vlottende activa / vreemd vermogen kort. De uitkomst hiervan moet minstens 2 zijn (streefgetal).

Quick ratio

(Vlottende activa – voorraden) / vreemd vermogen kort. De uitkomst hiervan moet minstens 1 zijn (streefgetal).

Werkkapitaal

Vlottende activa – vreemd vermogen kort. Dit is de marge die een bedrijf heeft om aan de korte termijnschulden te voldoen. De uitkomst hiervan moet positief zijn.

Solvabiliteit

Geeft inzage of de organisatie op lange termijn aan haar schulden kan voldoen. Kan de organisatie blijven bestaan?

Solvabiliteitsratio

Eigen vermogen / vreemd vermogen. Streefgetal = 100%

Debt ratio

Vreemd vermogen / totaal vermogen. De debt ratio geeft aan in welke mate een bedrijf is gefinancierd met vreemd vermogen. De uitkomst hiervan mag max 0,75 zijn (streefgetal). Hoe hoger hoe slechter de solvabiliteit.
