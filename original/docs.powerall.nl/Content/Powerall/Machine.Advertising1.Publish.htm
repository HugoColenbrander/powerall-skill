Advertentiekoppeling | Publiceren machine

Om een machine te publiceren via de advertentie koppeling, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud machines (MBB).

    1. Koppel de juiste machine/artikelgroep en configuratie aan de machine, die doorgeplaatst moet worden.

    2. Ga naar tabblad Machine-configuratie.

      • Vul / controleer de configuratie.

    3. Ga naar tabblad Website

      of Commercieel bij instelling machine administratie 2.0.

      • Vink aan op welke websites de machine getoond moet worden.

      • Pas eventueel het volgende aan.

        • Prijstype (standaard vraagprijs).

        • Prijs weergave (standaard excl. BTW)

    Opmerking: Stap b en c kan ook de Powerall CRM app.

  2. Onderhoud machine:

    Klik hier voor meer informatie over belangrijke machine gegevens:

    Onderstaande punten zijn belangrijk voor de advertentie:

    1. Machine merk + type + bouwjaar.

    2. Hoofd- en subgroep van de machine.

    3. Tabblad Machine Configuratie:

      1. Configuratie (PWC-configuratie).

    4. Tabblad Financieel:

      1. Adviesprijs (= standaard vraagprijs).

      2. Evt. verkoper.

    5. Tabblad Service:

      1. Tellerstand 1.

    6. Tabblad Website:

      1. Vink de gewenste websites aan.

      2. Prijstype (standaard Vraagprijs).

      3. Omschrijving.

      4. Uitvoering.

      Hiierbij een voorbeeld:

      1. Vink de websites aan, waarop de machine gepubliceerd moet worden.

      2. Voeg indien gewenst een extra omschrijving toe.

      3. Voeg indien gewenst een extra omschrijving van de uitvoering toe.

      4. Selecteer het prijstype (afhankelijk hiervan wordt de vraagprijs of bijv. de prijs op aanvraag getoond).

      5. Wijzig eventueel de prijsweergave exclusief / inclusief BTW.

      6. Voeg eventueel websitefoto’s toe (deze worden ook voor de Word-offertes gebruikt).

        • Er bestaat voor het toevoegen van websites ook een work-around, hiermee kan je foto’s snel toevoegen, zonder dat je ze 1 voor 1 moet selecteren.
        • Klik hier voor meer informatie om dit te doen:

          Als de foto’s in dossiermap van het servicenummer staan, doorloopt u de volgende stappen:

          1. Ga naar tabblad Dossier.

          2. Open de dossier map.

          3. Selecteer al de foto's voor de website.

          4. Rechtermuisknop.

          5. Wijzig de naam in <servicenr>-001_ bijv. 3-001_.

            • De foto's krijgen dan automatisch een volgnummer er achter.

          6. Ga naar tabblad Website.

          7. Kies voor de knop  Foto's.

            1. Het scherm Website foto's verschijnt.

            2. Wijzig eventueel de volgorde d.m.v. de pijltjes.

            3. Kies voor de knop Opslaan.

    7. Via het Mijn Powerall / klantenportaal kan er in de gaten gehouden worden of en welke machines (goed of met fouten) geplaatst worden.

      Opmerking: Al deze info kan ook gevuld worden vanuit de Powerall CRM app. Met name websitefoto’s toevoegen vanuit de CRM-app gaat gemakkelijk.

  3. De machine wordt automatisch doorgezet.

    • De machines worden elk uur opgehaald 3:11, 4:11 .. .7:11, 8:11 etc .. t/m 22:11u (dus 20 x), behalve tussen 22:11 en 03:11.

      Er kan wat vertraging zitten tussen het moment van aanpassing in Powerall Connect en bijwerken op de websites.

  4. Op het Mijn Powerall / klantenportaal > ADVERTENTIES > Bekijk machine overzicht onder "Bekijk machine overzicht' kan er geraadpleegd worden, welke machine op welke site getoond wordt.

    1. Groen succesvol geplaatst. Als je drukt op deze status, kan er in sommige gevallen doorgeklikt worden naar de URL.

    2. Oranje succesvol geplaatst, maar met 1 of meer waarschuwingen (vaak onvolkomenheden in de machinegegevens). Klik door op de machine om te zien wat er mis is.

    3. Rood (nog) niet geplaatst. Klik door op deze machine om te zien wat er mis is.

    • Als je bij een machine doorklikt op een advertentie, kan je eventueel de rubriek wijzigen waar de machine getoond moet worden, als deze foutief blijkt te zijn, zie Hoe pas ik bij een machine rubriek op website?

  5. Op het Mijn Powerall / klantenportaal> ADVERTENTIEKOPPELING > WEBSITES > klik op een website, zijn de instellingen per website te beheren.

    Neem bij onduidelijkheden contact op met de helpdesk.

    Opmerking: De wijzigingen worden direct opgeslagen.
