Onderhoud HS-code / Customs code

Inleiding

De Harmonized System code (HS-code), Taric-code of UCC (Union Customs Code) is een uitbreiding op de CBS-code en is 10 posities lang.

Voor meer informatie m.b.t. doel en instellingen zie:

  • Onderhoud CBS-codes
  • Wordt gebruikt voor de export / douane aangifte.
  • De term HS-code wordt gebruikt i.p.v. customs-code. Het gebruik hiervan kan al of niet ingesteld worden, vanaf versie 3.19.
    • Bij Instellingen artikelen, tabblad Scherminvoer is het veld HS-code in te stellen.
    • Hierdoor kan het veld HS-code bij Onderhoud artikelen ingegeven worden.
  • Wordt gebruikt bij Onderhoud artikelen als bedrijfsland niet Nederland (NL) is.

Opmerking: Deze opgave verschijnt als CBS-codes gebruikt worden en de (ISO) landencode bij Instellingen bedrijf niet NL is. Dit programma staat standaard in het menu vanaf versie 3.19.

Onderhoud

Om een HS-code toe te voegen of te wijzigen, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud HS-code (ABD).

    • In Menu 3.0 onder Artikelen > Overige > Configuratie.

  2. Geef de HS-code in.

    • Bij F4 verschijnt het scherm Zoek customs-codes.

  3. Kies voor de knop  Wijzigen.   of   Kies voor de knop Opvoer.

  4. Geef de afkorting, omschrijving en externe omschrijving op.

  5. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

 

Onderhoud artikelen

Bij Onderhoud artikelen kan bij een artikel ook de HS-code ingegeven worden instelling.

NB De HS-code kan op de (kopie) factuur, offerte of overzichten weergegeven worden.
