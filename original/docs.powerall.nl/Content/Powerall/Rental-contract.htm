Verhuur overdracht

Bij Onderhoud relaties (BRR), tabblad E-document kan ingesteld worden bij 17 Verhuuropdracht + E-MAIL / PDF dat er een e-mail verzonden wordt, bij een overdracht.

Wat wordt er bij de verhuuroverdracht bijgewerkt?

Het volgende wordt bijgewerkt a.d.h.v. meegegeven overdrachtnummer:

  • Machine verkoopstatus, bijgewerkt als deze retour is gekomen.

  • Machine tellerstand

  • Status van overdracht

  • Contractregels status

  • Contractregels retourdatum + tijd OF leverdatum + tijd (afhankelijk van regeltype)

  • Voorraad accessoires

  • Artikelen bijwerken e.d.
