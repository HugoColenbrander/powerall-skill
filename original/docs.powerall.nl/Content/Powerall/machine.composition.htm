Machine samenstelling

Doel

Met deze functie is het mogelijk om een koppeling te maken tussen de verschillende servicenummers.

  • Hierbij is te zien uit welke (sub) machines een bepaalde machine bestaat.

  • Daarnaast kan op de verschillende 'onderdelen' van een machine, onderhoud gedaan worden.

Waar kan ik de samenstelling ingeven?

Bij Onderhoud machines (MBB) is mogelijk om het veld Onderdeel van in te geven.

  • Onderdeel van Geef hier het servicenummer van de hoofdmachine in, waarvan deze een onderdeel is instelling.

Waar zie ik de machine samenstelling?

Bij de 'hoofd' machine verschijnt de knop Samenstelling bij:

  • Onderhoud machines (MBB).

  • Opvragen machines (MO).

  • De machine samenstelling wordt hierbij getoond.

    • Samenstelling Hiermee kan de Machine samenstelling bekeken worden instelling.

    Die bestaat uit <merk> <type> en < omschrijving 1>.

    • Het servicenummer kan aan maar één machine gekoppeld zijn.
    • Het gekoppelde servicenummer mag zelf niet gekoppeld zijn (maximaal 2 niveaus).
      • Anders verschijnt de melding Dit servicenummer is al onderdeel van een andere machine <servicenummer>.

FAQ

  • Bij een in- of verkoop moet de samenstelling handmatig bijgewerkt worden.
  • Beschikbaar vanaf versie 3.13.
Wat moet er ingesteld zijn in Powerall?

Bij Instellingen machines (BIS), tabblad Scherminvoer moet het veld Gebruik machine samenstelling op Ja staan:

  • Gebruik machine samenstelling Optie voor het gebruik van een machine samenstelling.
    - Nee (standaard)
    - Ja
    Bij Ja komt het veld Onderdeel van beschikbaar, hierbij kan een 'hoofd' machine ingegeven worden. Bij deze machine(s) komt de knop Samenstelling beschikbaar.
