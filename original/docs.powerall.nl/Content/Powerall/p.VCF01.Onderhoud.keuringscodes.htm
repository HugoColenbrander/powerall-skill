Onderhoud keuringscodes

De keuringscodes worden gebruikt om bepaalde keuringen uit te voeren. Om deze toe te voegen of te wijzigen, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud keuringscodes (MKB).

  2. Geef de keuringscode in.

    • Bij F4 verschijnt het scherm Zoek keuringscodes.

  3. Kies voor de knop  Wijzigen.   of   Kies voor de knop Opvoer.

  4. Selecteer het keuringstype:

    • Eigen keur

    • APK keuring

    • Tachograaf keuring

    • VA-Keur

      Opmerking: Bovenstaand de VA-Keur standaard instellingen.

  5. Vul de gegevens in of wijzig deze.

  6. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

 

 

FAQ

Hieronder een uitleg van de velden van scherm Onderhoud keuringscodes:

 

Keuringscode De code van de keuring.
 
Afkorting De afkorting van deze code.
Omschrijving De 'betreffende' omschrijving.
 
Keuringstype Optie voor het type keuring / keuringtype:
- VA-Keur (hierbij verschijnen de VA-velden)
- APK keuring
- Tachograaf keuring
- Eigen keur, kan opgegeven worden bij Onderhoud formulieren.
 
Laatste import (VA-Keur) De laatste datum van de VA-Keur import.
Import communicatiecode De communicatiecode Import, deze moet aanwezig zijn voor VA-Keur bij Instellingen externe communicatie.
Rapport communicatiecode De communicatiecode voor rapporten. Is niet verplicht, bij ingave moet deze aanwezig zijn bij Instellingen externe communicatie.
Export communicatiecode De communicatiecode voor de Export, deze moet aanwezig zijn bij Instellingen externe communicatie.
Logging bewaren Optie om de log te bewaren:
- Ja (standaard)
- Nee

Opmerking: Bij ja / VA-Keur kan de log meer dan 1 GB groot zijn!
Export bestandspad In dit bestandspad wordt de download/export opgeslagen, geldt o.a. voor:
- Download VA-Keur
- Bestandspad Export
Keuringsveld element 1 Element 1 voor de keuring, gebruikt voor VA-Keur.
- MachineType (standaard VA-Keur).

Synchronisatie machines (VA)  
Synchronisatie methode Optie hoe er gesynchroniseerde wordt:
- Eenmalig (standaard)
- Continue
Id nr. eigenaar nummer Optie waaraan de eigenaar gekoppeld is voor gesynchroniseerd (optioneel):
