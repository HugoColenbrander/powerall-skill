Connect API logging

Om de API logging te bekijken, doorloopt u de volgende stappen:

  1. Ga naar Connect API logging (BUC).

    • In Menu 3.0 via Systeem > Overige > Audit trail.

  2. Maak een selectie.

  3. Kies voor de knop  Selecteer.

    • Indien aanwezig, verschijnen de gegevens op het scherm.

    • Vanaf versie 3.27 is de logging verbeterd en wordt weergegeven wat er precies gewijzigd is.

  4. Kies voor de knop  Afsluiten.

  • Als de kolom Bestand gevuld is, wordt de regel als een soort foutregel gekenmerkt en in het rood weergegeven.
  • I.c.m. de API Gereserveerd en Picken wordt het aantal weergegeven, vanaf versie 3.21.
Welke Powerall data kan gevuld worden vanuit een API?

De volgende gegevens kunnen toegevoegd of bijgewerkt worden vanuit een API:

  • Nieuwe relatie aanmaken / wijzigen

  • Nieuw artikel aanmaken / wijzigen / leverancier artikel

  • Nieuw machine aanmaken of kopiëren / service object / wijzigen

  • Selecteren inkoopfacturen

  • Nieuwe inkooporder aanmaken

  • Kas of bank boeken

  • Overzicht orders (te picken) 2.0 d.w.z. verkooporderregel aantallen gereserveerd/gepickt verwerken.

  • Nieuwe verkooporder aanmaken

    • I.c.m. servicenummer wordt ook een machine transactie aangemaakt.

  • Nieuwe servicemelding aanmaken

  • Nieuwe werkorder aanmaken

Voor meer info zie:

  • developers.powerall.nl
