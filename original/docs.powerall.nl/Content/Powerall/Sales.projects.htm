Verkoopprojecten

Bij een verkoopproject is het mogelijk om de kosten en opbrengsten aan een bepaald project te koppelen, zodat je inzicht hebt wat er daadwerkelijk verdiend is.

Inleiding

Zie hieronder de werking voor de verkoop projecten module binnen Powerall Connect.

Opmerking: Let op, dit is geen productie handleiding, maar de verkoop projecten optie.

Verkoopprojecten

Verkoop project

Om een nieuwe verkoopproject aan te maken, doorloopt u de volgende stappen:

  1. Ga naar Selecteren projecten (PS).

  2. Kies voor de knop  Nieuw. 

    • Het scherm Invoeren/wijzigen projecten verschijnt.

    1. Tabblad Algemeen:

      • Kies een klant en enter verder naar het project.

      • Vul de leverdatum in.

      • Laat de projectstatus op vrij staan.

      • Stel het projectsoort in op verkoop.

      • Eventueel kan er een afgesproken bedrag ingevoerd worden (dit is alleen informatief).

  3. Maak een nieuwe fase aan door aan de linkerkant met de rechter muisknop op project te klikken, en dan Nieuwe fase.

    1. Vul hier de standaard fasecode in, en laat de status in eerste instantie op Engineering staan. Fases van een verkoopproject zijn bijv. : Stalinrichting; Melk robot; Melktank.

    2. Als alle fases erin staan moet de projectstatus gewijzigd worden van vrij naar actief.

    3. Hierna moeten de verschillende fases veranderd worden van status Engineering naar Actief. Dit kan d.m.v. rechter muisknop op de fase, wijzigen fase en daar de status omzetten.

  4. Via het tabblad Realisatie kunnen er nieuwe orders gekoppeld worden, maar ook lopende orders (d.m.v. knop Functies) en zelfs vrijgegeven orders.

    Belangrijk:  Let op offertes kunnen pas gekoppeld worden als de projectstatus vrij is.

    • Het koppelen kan ook andersom, vanuit bijv. een werkorder, of verkooporder, onder de knop > Meer functies.

  5. Inkoopfacturen kunnen via de juiste rekeningen (d.m.v. de optie kostendrager: projecten) op het project geboekt worden.

Dossier:

  • In tabblad Dossier, kan technische documentatie / foto’s van dit project geplaatst worden. Deze informatie wordt in het relatiedossier van de klant opgeslagen.

Afwerken:

  • Als alle orders zijn vrijgegeven / afgewerkt en de inkoopfacturen ingeboekt zijn, dan kan per fase de status van Actief gewijzigd worden naar Gereed.
    • Zijn alle fases gereed, dan kan het project op het tabblad Algemeen van project status Actief naar Gereed gezet worden.
  • Het resultaat van het project kan bekeken worden via het tabblad Realisatie (onderaan) of via de knop Realisatie aan de rechterkant van het scherm.

Opmerking: Bij de module projecten zijn beide opties verkoopproject of projecten/productie mogelijk, je kiest dan bij het aanmaken van het project de soort.

FAQ

Wat doe ik bij de melding 'Niet mogelijk om regels, waar geen referentie is ingevuld te selecteren'?

Deze melding wordt gegeven als het veld onder de kop "Project" niet ingevuld is.

  • In dit geval de skelter al aan een project gekoppeld, dus daar kan geen nieuw project voor opstarten worden.

Het gaat om deze melding:

Verkooporder > Projecten wordt gestart via de knop(pen): < Meer functies > Verkoopprojecten.
