Powerall Connect API

  • developers.powerall.nl

    • Via ../developers.powerall.nl/interface

De volgende API's zijn beschikbaar:

  • Leveringen / Deliveries

FAQ

 

Opmerking: Bent u op zoek naar een API-key? Neem hiervoor contact op met uw Powerall leverancier.

Ik krijg een foutmelding, wat moet ik doen?

Bij een foutmelding wordt er een status teruggegeven. De status is afhankelijk van de actie:

  • Status Omschrijving Actie
    400- 499 De invoer is niet juist, fout van de gebruiker. Verbeter de request.
    500 Fout in de API. Neem hiervoor contact op met de helpdesk.
Wat is een API:

API is een afkorting en staat voor Application Programming Interface.

Een API is een klein stuk software en zorgt er voor dat twee verschillende programma's of systemen met elkaar kunnen communiceren.

Worden webhooks ondersteund?

M.a.w. kan Powerall bij elke wijziging een seintje sturen naar het opgegeven endpoint?

  • Nee

Zit er een rate limiting op de API?

M.a.w. is er een maximaal aantal aanvragen per minuut / dag.

  • Nee, er zit geen limiet hier op.
