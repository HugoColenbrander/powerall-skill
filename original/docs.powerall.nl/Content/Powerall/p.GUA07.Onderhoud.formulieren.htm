Onderhoud PDF formulieren (per werkorder)

Onderhoud formulieren wordt o.a. gebruikt voor de VA-Keur. Ook kan een PDF sjabloon document worden opgegeven, die gekoppeld kan worden aan een machine. Met de Werkbon App kan dit formulier worden opgeroepen en ingevuld.

Om deze toe te voegen of te wijzigen, doorloopt u de volgende stappen:

  1. Ga naar Onderhoud formulieren (BAF).

  2. Geef de formuliercode in.

    • Bij F4 verschijnt het scherm Zoek formulieren.

  3. Kies voor de knop  Wijzigen.   of   Kies voor de knop Opvoer.


  4. Wijzig de instellingen.

    • Het type kan normaal of keuring zijn, selecteer het sjabloon.

  5. Kies voor de knop Opslaan.

De gegevens zijn opgeslagen.

 

FAQ

Hieronder een uitleg van de velden van scherm Onderhoud formulieren:

 

Formuliercode Dit is de afkorting van deze code.
Naam Dit is de naam.
Type Het type formulier:
- Normaal (standaard) altijd beschikbaar in de Werkbon App dus niet afhankelijk van bijv. een werkorder met een keuring.
- Keuring, hierbij is de keuring(scode) verplicht.
Sjabloon Dit is het betreffend sjabloon.
 
Keuring Beschikbaar bij type keuring:
Layout Dit is de lay-out van de keuring.
- A (standaard) bij type is keuring
- Leeg (standaard) bij type is normaal
NB Bij een keuring moet deze A layout aanwezig zijn bij VCF71 met bijv. het veld Tellerstand 1 MVM120.
Keuring De keuringscode, zie Onderhoud keuringscodes.

 

Hoe koppel ik de formulier- of keuringscode aan een machine?

Bij Onderhoud machines (MBB) Tabblad Service  onder het kopje Keuringsinfo  kan de formulier (code) ingegeven worden. Om een keuring te selecteren bij een werkorder, doorloopt u de volgende stappen:

  1. Ga naar Werkorder wijzigen.

  2. Kies voor de 3 puntjes knop, achter het servicenummer.

    1. Het scherm Transacties verschijnt.

    2. Selecteer bij type transactie de keuring.

    3. Wijzig eventueel de keuringsdata.

    4. Kies voor de knop Opslaan.

      1. Onder de knop Dossier staat een PDF-formulier klaar, waarin de werkorder en machinegegevens al gevuld zijn.

      2. Belangrijk: De keuringsdatum en geplande keuringsdatum worden bijgewerkt in het formulier op het moment dat de werkloon gereed gemeld wordt op de Werkbon App.

De gegevens zijn opgeslagen.
