Machine offerte

Voor de algemene machine offertes, doorloopt u de volgende stappen:

  1. Ga naar Machine offertes (OS).

    • Het scherm Selecteren offertes verschijnt

  2. Voor meer informatie zie Selecteren offertes.

 

Machine offertes i.c.m. de motorbranche

Belangrijk:
• Alleen beschikbaar als bij Instellingen bedrijf soort bedrijf motorbranche is.
• Bij Instellingen machines moet het artikel voor machineverkoop gevuld zijn.

Het is mogelijk om vanuit selecteren machines een offerte aan te maken. Om dit te doen, doorloopt u de volgende stappen:

  1. Ga naar Selecteren machines (MS).

  2. Ga naar tabblad Beschikbaar.

  3. Selecteer de machine in de regels.

  4. Kies voor de knop  Offerte. instelling

    • Het scherm Offerte aanmaken verschijnt.

  5. Geef de relatiecode in.

    Opmerking: Bepaalde RDW gegevens, zoals BPM, worden opgehaald a.d.h.v. het kenteken.

  6. Geef de prijs incl. of excl. BTW in.

  7. Bij in te ruilen machines optioneel.

    • Selecteer de in te ruilen machine.

  8. Bij Inruil

    1. Selecteer de inruil Machine.

    2. Controleer / wijzig de gegevens.

  9. Kies voor de knop  Aanmaken.

    • Het scherm Invoeren/wijzigen offertes verschijnt.

De offerte is toegevoegd.
