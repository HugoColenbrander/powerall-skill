Werkorder wijzigen

Het wijzigen van een werkorder kan vanuit verschillende schermen in Powerall Connect:

  • Direct via de Werkorder module.

  • Vanuit een servicemelding via de Servicemeldingen module.

Via werkorders

  1. Ga naar Selecteren werkorders (WS).

  2. Selecteer de werkorder (WO).

  3. Kies voor de knop  Wijzigen.

    • Het scherm Invoeren/wijzigen werkorders verschijnt.

  4. Wijzig de werkorder gegevens.

    Opmerking: Het is mogelijk om een werkorder te wijzigen en tegelijk bij urenregistratie in- en uit te klokken op dezelfde werkorder.

  5. Kies voor de knop  Einde order.

Via servicemeldingen

  1. Ga naar Selecteren servicemeldingen (SS).

  2. Selecteer de servicemelding.

    Tip: In de kop kunt u zoeken/selecteren op werkorder-, servicenummer, relatie, etc.

  3. Kies voor de knop  Beheer.

    • Het scherm Beheer servicemelding verschijnt.

    • Rechtsboven staan de (openstaande) werkorders. De WO wordt ook weergegeven in het onderste vak bij tabblad Planning als deze ingepland staan.

  4. Selecteer de werkorder.

  5. Kies voor de knop  Wijzigen.


    Het scherm Invoeren/wijzigen werkorders verschijnt.

  6. Wijzig de werkorder gegevens.

  7. Kies voor de knop  Einde order.

De gegevens zijn opgeslagen.
