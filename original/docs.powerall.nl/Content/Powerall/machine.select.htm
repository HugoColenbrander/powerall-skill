Selecteren machines

Met behulp van dit programma is het volgende mogelijk bij machines:

  • Uitgebreide filter of selectie mogelijkheden.
    • Selectie op inkoop- en verkoopstatus incl. verhuur/contract machines
  • Sortering op kolommen
  • Machine direct te wijzigen
  • Te ontvangen
    • Boeken goederenontvangst
  • Voor het bijwerken van de keuringsdatum zie: Hoe werk ik de APK of Tachograaf datum bij?

Om alle of bepaalde machines te bekijken, of te wijzigen, doorloopt u de volgende stappen:

  1. Ga naar Selecteren machines (MS).

  2. Geef een selectie op.

    Uitgebreidere selectie, vanaf versie 3.20.

    Kies voor de knop  Selecteren.

  3. Selecteer een machine.

  4. Kies voor de knop  Wijzigen.

    1. Wijzig de machine.

    2. Kies voor de knop Opslaan.

    De machine regel wordt bijgewerkt.

  5. Kies voor de knop  Nieuw.

    • Zie Nieuw machine aanmaken of kopiëren.

  6. In tabblad Te ontvangen is het mogelijk om een machine te ontvangen.

    De machines met de inkoopstatus besteld of inruil (verwacht) worden weergegeven.

    1. Kies voor de knop  Ontvangen.

      Alleen beschikbaar bij gesloten machine administratie.

      • Het scherm Openstaande inkooporders verschijnt.

        Opmerking: Er moet wel een openstaande inkooporder aanwezig zijn.

    2. Kies voor de Enter-toets of dubbelklik op de inkooporder om deze te boeken.

      • Van de inkooporder moet een bestelbon afgedrukt zijn, anders kan deze niet geboekt worden.

    3. De machine kan geboekt worden, zie Boeken goederenontvangst.

      • Bij machinegegevens kan het serienummer / kenteken ingegeven / gecontroleerd worden.

      • De inkoopstatus wordt voorraad.

        • De machine moet nog geleverd worden, zie tabblad Leveren.

  7. In tabblad Beschikbaar is te zien of de machine een dossier heeft.

    1. De machines met inkoopstatus besteld of voorraad worden getoond.

  8. In tabblad Te leveren zijn de machines te zien, die nog geleverd moeten worden.

    • Kies voor de knop  Leveren.

      Alleen beschikbaar bij gesloten machine administratie.

      1. Het melding Machine leveren verschijnt.

        • Kies voor de knop  OK.
      2. Het scherm Machinegegevens verschijnt.
        • Geef het serienummer / kenteken in of controleer deze.
      3. Kies voor de knop Opslaan.
        • De melding Leverbon printen verschijnt.

        De melding De machinetransactie is aangemaakt en succesvol verwerkt.

  9. In tabblad Verhuur/contracten zijn alle verhuur machines te zien.

    1. De verkoopstatus is hierbij Verhuur. Als de machine 'lopend' is, verschijnt de status Verhuurd.

    2. Het is mogelijk om een datum in te geven m.b.t. einddatum van de verhuur.

      • Hierbij wordt rekening gehouden met verhuur 2.0.

  10. In tabblad Service zijn de machines te zien, waaraan een keuring gekoppeld is.

    • D.w.z. er is een geplande keuringsdatum ingevuld.

    • Er kan een keuring geselecteerd worden.

    • D.m.v. de knop Keuringsdatum kunnen de APK of Tachograaf geplande datums bijgewerkt worden.

      • Zie ook: Hoe werk ik de APK of Tachograaf datum bij?

FAQ

Tip: Gebruik CTRL + F om een machine te zoeken.

  • Er kan ook gezocht worden op één element, op betreffende tabbladen, vanaf versie 3.21.
Hoe komt het dat ik niet alle machines direct zie?

Bij Selecteren machines wordt aan de hand van het vinkje Directe selectie de regels al of niet gevuld. Als het vinkje uit staan, kan eerst het volgende:

  • Een selectie ingegeven worden.

  • Naar een ander tabblad gegaan worden.

  • Bij veel machines kan het opvragen van alle gegevens in tabblad Alle kan enkele minuten duren.

Opmerking: Het vinkje en tabblad wordt opgeslagen en toegepast als het programma opnieuw uitgevoerd wordt.

Hoe kan ik een machine wijzigen?

Een machine kan gewijzigd worden via de knop Wijzigen. Om dit te doen, doorloopt u de volgende stappen:

  1. Geef eventueel een selectie in.

    • Kies voor de knop  Selecteer.

  2. Selecteer de machine, in de regels.

  3. Kies voor de knop  Wijzigen.

    • De machine kan gewijzigd worden.

Er wordt hierbij rekening gehouden met de autorisatie.

Wat doet het vinkje 'Direct selecteren'?

Als het vinkje Direct selecteren aangevinkt is, worden alle regels direct gevuld, als het programma opstart.

  • Hierdoor is de knop Selecteren niet meer nodig.

  • Als er steeds een selectie ingegeven wordt, is het beter om deze uit te vinken.

Wat doe ik bij de melding: 'Er zijn meerdere dossiermappen voor deze machine'?

Er zijn meerdere mappen met het <servicenummer> in de naam, bijv.:

  • Controleer, in de dossier map van de machines, de mappen en voeg de informatie samen in één map.

  • Opmerking: De melding Er zijn meerdere dossiermappen aanwezig, kan ook voorkomen bij artikelen, bij het gebruik van een underscore (laag liggend streepje) in de artikelcode.

Waar kan ik de machinegegevens / iconen m.b.t. website foto's en notities zien?

Alleen op tabblad Beschikbaar zijn de Website / Foto / Notitie iconen of symbolen zichtbaar.

Wanneer is op tabblad Te ontvangen de knop Ontvangen beschikbaar?

Als een machine de inkoopstatus Besteld of Inruil (verwacht) heeft is de knop Ontvangen beschikbaar, op tabblad Te ontvangen.

  • Ook verschijnt de vermelding Op inkooporder, als een machine op een inkooporder voorkomt.
Wanneer is op tabblad Te leveren de knop Leveren beschikbaar?

Als een machine de inkoopstatus Voorraad en verkoopstatus Verkocht heeft is de knop Leveren beschikbaar, op tabblad Te leveren.

En de administratie is een gesloten machine administratie, vanaf versie 3.05.

  1. Ook verschijnt er de melding Het te boeken bedrag voor de levering van de machine is x. xxx.xxx wilt u doorgaan.
  2. Kies voor de knop Ja.
