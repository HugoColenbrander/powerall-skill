Machine overboeken

Belangrijk: Dit geldt alleen bij een gesloten - en gedeelde machine administratie.

Bij meerdere administraties is het mogelijk om een machine over te boeken naar een andere administratie.

De voorwaarden hiervoor zijn o.a.:

  • De(ze) administratie moet de eigenaar zijn van machine (ook magazijn eigenaar).
  • Het van magazijn heeft als eigenaar deze administratie.
  • Het naar magazijn heeft een andere eigenaar als deze administratie
  • De machine mag geen status besteld of inruil hebben.

Om een machine over te boeken, doorloopt u de volgende stappen:

  1. Ga naar Overboeken machine (MVM).

  2. Geef het servicenummer in.

  3. Geef het <van> en <naar magazijn> in.

  4. Geef de overige transactiegegevens in.

    Opmerking: Het artikel bij overboeking moet een machine verkoopartikel zijn.

  5. Kies voor de knop Opslaan.

    • De melding Weet u zeker dat u deze machine wilt overboeken verschijnt.

    • Bij Ja verschijnt de melding Machine succesvol overgeboekt!

De gegevens zijn opgeslagen.

 

FAQ

  • Bij Opvragen machines (MO), tabblad Transacties is de overboeking te zien als Correcties aangevinkt is.
  • Bij tabblad Financieel staat bij de Inkoopgegevens het (naar) magazijn.
  • Bij geen gedeeld machine bestand verschijnt de melding: Programma slechts te gebruiken bij een gedeeld machinebestand.
  • Standaard staat dit programma niet in het menu.
