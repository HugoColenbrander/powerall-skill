Exporteren gegevens

Met behulp van dit programma kunnen verschillende gegevens geëxporteerd worden naar een .txt bestand. Om dit te doen, doorloopt u de volgende stappen:

  1. Ga naar Exporteren gegevens (BEE).

  2. Geef de layoutcode in.

    • Bij F4 verschijnt het scherm Zoek export layouts.

  3. Geef een selectie op indien nodig.


    Opmerking: De bestandsnaam kan bij Onderhoud export layouts (BEL) gewijzigd worden. De overig velden zijn hier ook gedefinieerd.
    • De laatste 3 velden zijn selectievelden, die gedefinieerd zijn in de export layoutcode.

  4. Kies voor de knop  Exporteren.

De gegevens zijn geëxporteerd.
