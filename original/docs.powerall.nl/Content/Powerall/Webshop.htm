FAQ

Kan ik ook een YouTube filmpje toevoegen op mijn website?

Ja, dit is mogelijk om en YT URL (verwijzing naar een webpagina) mee te geven, mits deze gekoppeld is aan een artikel, bijv. artikel BR960005038.

Welke gegevens kan ik allemaal koppelen met mijn webshop?

De volgende gegevens kunnen via een webshop gekoppeld worden:

  • developers.powerall.nl - nieuw -

Deze dienst kan door Powerall klanten gebruikt worden, om relevante informatie te verkrijgen voor:

  • het runnen van een webshop,

  • om bestellingen te plaatsen en

  • om relaties aan te gaan.

Zoek (artikel) woorden webshop

De zoekwoorden van de Artikelomschrijvingen worden meegenomen bij de webshop in het veld WebShopSearchText bij een ProductGet.

De artikel kenmerken staan in het veld Properties.
