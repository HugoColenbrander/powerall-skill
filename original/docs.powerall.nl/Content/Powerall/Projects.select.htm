Voor het dashboard projecten 2.0, doorloopt u de volgende stappen:

  1. Ga naar Selecteren projecten (PS).

    • Alle openstaande projecten verschijnen op het scherm.

  2. De volgende acties zijn mogelijk:

    1. Geef een selectie op.

      • Kies voor de knop  Selecteer.

    2. Kies voor de knop  Planningsvoorstel.

      • Het scherm Projectplanningslijst verschijnt.

    3. Kies voor de knop  Nieuw.

      • Nieuw project aanmaken

    4. Kies voor de knop  Wijzigen.

    5. Kies voor de knop  Details.

    6. Kies voor de knop  Bestelgegevens.

      • Het scherm Zoek bestelgegevens verschijnt, met de wel/niet bestelde artikelen.

    7. Kies voor de knop  Print picklijst.

      • Het scherm Overzicht project picklijst verschijnt.
